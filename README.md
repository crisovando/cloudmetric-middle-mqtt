# Iniciar el proyecto

Para iniciar el proyecto, sigue estos pasos:

1. Instala Rust: Visita https://rustup.rs/ y sigue las instrucciones.
2. Clona el repositorio: `git clone <url-del-repo>`
3. Navega al directorio del proyecto: `cd middle-mqtt`
4. Construye el proyecto: `cargo build`
5. Ejecuta el proyecto: `cargo run`
